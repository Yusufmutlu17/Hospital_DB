# 🏥 Hastane Yönetim Bilgi Sistemi

## 📖 Proje Hakkında

Bu proje, **Veritabanı Sistemleri** dersi kapsamında geliştirilmiş bir **Hastane Yönetim Bilgi Sistemi**dir. Sistem; hasta, doktor, poliklinik, randevu ve muayene süreçlerini ilişkisel veritabanı yapısı kullanarak yönetmek amacıyla tasarlanmıştır.

Veritabanı tasarımı **Oracle Live SQL** kullanılarak gerçekleştirilmiş olup proje kapsamında ER diyagramı, ilişkisel şema ve SQL sorguları oluşturulmuştur.

## ✨ Özellikler

- Hasta ve doktor kayıtlarını yönetme
- Poliklinik bilgilerini görüntüleme
- Hasta randevularını oluşturma ve takip etme
- Doktor-hasta muayene kayıtlarını saklama
- İlişkisel veritabanı yapısıyla sorgulama işlemleri gerçekleştirme

## 🛠 Kullanılan Teknolojiler

- Oracle Live SQL
- İlişkisel Veritabanı Tasarımı
- ER Diyagramı

## 🎯 Proje Amacı

Bu proje, hastane yönetim süreçlerini ilişkisel veritabanı mantığıyla modelleyerek SQL ve veritabanı tasarımı konularındaki bilgi ve becerileri uygulamalı olarak geliştirmeyi amaçlamaktadır.
